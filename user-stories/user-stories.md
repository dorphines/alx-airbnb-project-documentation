# Airbnb Clone User Stories

This file contains user stories derived from the use case diagram, capturing the key interactions and motivations of the users.

## User Stories

1.  **As a Guest, I want to be able to register for an account so that I can book properties.**
    -   **Acceptance Criteria:**
        -   I can sign up using my email and a password.
        -   I receive a confirmation email after registration.
        -   My password is securely stored.

2.  **As a Guest, I want to search for properties by location and date so that I can find a place to stay.**
    -   **Acceptance Criteria:**
        -   I can enter a city or address to search for.
        -   I can select a check-in and check-out date.
        -   The search results show only available properties that match my criteria.

3.  **As a Host, I want to list my property with details and photos so that I can attract potential guests.**
    -   **Acceptance Criteria:**
        -   I can fill out a form with property details (title, description, price, amenities).
        -   I can upload multiple photos of my property.
        -   I can set the availability of my property on a calendar.
        -   My property listing is visible in search results after creation.

4.  **As a Host, I want to manage my bookings so that I can see who is staying at my property and when.**
    -   **Acceptance Criteria:**
        -   I have a dashboard to view all my upcoming and past bookings.
        -   I can see the guest's name and the dates of their stay for each booking.
        -   I have the option to approve or decline booking requests.

5.  **As a Guest, I want to leave a review for a property I stayed at so that I can share my experience with other users.**
    -   **Acceptance Criteria:**
        -   I can only review a property after my booking is complete.
        -   I can provide a rating (e.g., 1-5 stars) and a text comment.
        -   My review is visible on the property's listing page.

6.  **As an Admin, I want to be able to manage users so that I can maintain a safe and trustworthy platform.**
    -   **Acceptance Criteria:**
        -   I can view a list of all users.
        -   I can suspend or delete a user's account.
        -   I can filter users by their role (Guest, Host).
